A modern, responsive, and animated user interface for the GatePass System. Designed for ease of use and visual excellence.

## ✨ Highlights

- **⚡ Blazing Fast**: Powered by Vite for near-instant development and optimized builds.
- **🎨 Premium UI**: Smooth animations using Framer Motion and beautiful icons from Lucide.
- **📱 Responsive Design**: Fully adaptible to mobile, tablet, and desktop screens.
- **🔄 Dynamic Flow**: Real-time updates and interactive material pass tracking.
- **🛠️ Robust Logic**: Custom hooks and modular components for scalability.

## 🛠️ Tech Stack

- **Library**: React 18
- **Build Tool**: Vite
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router DOM (v6)
- **API Client**: Axios

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   cd frontend
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure Environment**
   Ensure you have a `.env` file with your backend URL:
   ```env
   VITE_API_URL=http://localhost:5000/api
   ```

4. **Launch Dev Server**
   ```bash
   npm run dev
   ```
   *The app will be available at `http://localhost:8000`*
